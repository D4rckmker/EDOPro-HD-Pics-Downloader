# Instalación de Python (macOS)

[Read in English](PYTHON_INSTALL.md)

**¿No tienes Python instalado?** Sigue esta guía rápida.

## Opción A — Instalador oficial (recomendado)
1. Ve a: https://www.python.org/downloads/
2. Descarga la última versión de **Python 3** para macOS.
3. Abre el `.pkg` e instala.
4. Verifica en Terminal:
   ```bash
   python3 --version
   ```

## Opción B — Homebrew
Si ya tienes Homebrew:
```bash
brew install python
python3 --version
```
